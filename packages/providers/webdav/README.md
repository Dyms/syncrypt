# @syncrypt/provider-webdav

WebDAV `StorageProvider` (e.g. Nextcloud). Implements the universal object subset
(`put`/`get`/`list`/`delete`/`stat`) and reports `capabilities()` honestly.
Passes the shared provider conformance suite.

Spec: [RFC-0006](../../../docs/rfc/RFC-0006-Storage-Provider-API.md).
Status: planned (post-1.0) — see [ROADMAP](../../../ROADMAP.md).
