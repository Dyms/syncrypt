# @syncrypt/sdk

Public API that wires a concrete provider + vault adapter + crypto into a
ready-to-use `SyncEngine` (`push`, `pull`, `sync`, `dryRun`, `status`). Consumed by
clients such as the Obsidian plugin (or a future CLI).

Spec: [RFC-0003](../../docs/rfc/RFC-0003-Architecture.md).
Status: not yet implemented — see [ROADMAP](../../ROADMAP.md).
