# Syncrypt

> Перевод. Источник истины — [English README](./README.md). При расхождении верна английская версия; перевод может отставать.

> Простая, безопасная и предсказуемая синхронизация Obsidian — данные принадлежат тебе.

**Syncrypt** — небольшой, объяснимый движок синхронизации для хранилищ (vault)
[Obsidian](https://obsidian.md). Он держит твои заметки одинаковыми на macOS,
Windows и Android, используя хранилище, **которым владеешь ты** (сначала
S3-совместимое; другие провайдеры позже), с **клиентским сквозным шифрованием**.

Это сознательно *не* инструмент совместного редактирования в реальном времени. Он
делает одно и хорошо: переносит твои Markdown-файлы между твоими устройствами и
твоим хранилищем — безопасно и так, что это всегда можно понять и починить руками.

## Зачем это нужно

Синхронизация на CRDT в реальном времени (например, Self-hosted LiveSync) мощная,
но решает *другую* задачу. Для одного пользователя, который не редактирует одну и ту
же заметку на двух устройствах одновременно, скрытая база данных, дерево ревизий и
автоматическое разрешение конфликтов добавляют риск (тихое дублирование, потеря
данных при повреждении), не давая ничего взамен.

Syncrypt делает ставку наоборот:

- **Источник правды — твои Markdown-файлы**, а не база данных.
- Синхронизация — это просто **upload** и **download**, скоординированные обычным
  JSON-манифестом.
- **Никакой магии**: каждое действие объяснимо одной фразой
  (*«удалённая версия новее»*, *«локальный хеш отличается от манифеста»*,
  *«файл помечен как удалённый»*).
- Если что-то пошло не так — любой файл можно открыть, восстановить или скачать
  вручную.

## Принципы

- **Simple. Secure. Predictable.** (Просто. Безопасно. Предсказуемо.)
- **Данные принадлежат пользователю** — без vendor lock-in, без проприетарного
  формата, без скрытой базы.
- **Никакой магии** — каждый шаг синхронизации объясним.
- **Offline first** — потеря сети не событие.
- **Markdown first** — простые файлы это контракт.
- **Ноль телеметрии** — Syncrypt никогда не «звонит домой».

Главная директива: **Syncrypt не должен удивлять пользователя.**
