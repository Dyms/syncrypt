# Быстрый старт

> Перевод. Источник истины — [English](../../user-guide/getting-started.md). При расхождении верна английская версия.

## Что нужно

- Obsidian на каждом устройстве (macOS / Windows / Android).
- Хранилище под твоим контролем: S3-совместимый бакет **или** WebDAV-папка —
  см. [установку и настройку](../install.md).
- Надёжная **passphrase**, которую ты не забудешь. **Если потеряешь её —
  данные не восстановить, это заложено в дизайне.**

## Первое устройство

1. [Установи Syncrypt через BRAT](../install.md) и открой настройки плагина.
2. Введи данные хранилища (endpoint, бакет/папка, префикс, ключи).
3. Выполни **Syncrypt: Unlock** и задай **passphrase**. Ключи выводятся
   локально и не покидают устройство; в хранилище уходят только несекретные
   параметры, чтобы другие устройства могли вывести те же ключи.
4. При желании настрой **профиль синхронизации** (что синхронизировать) — см.
   [конфигурацию](./configuration.md). По умолчанию — все заметки и вложения.
5. Выполни **Sync now** — первая зашифрованная загрузка.

## Остальные устройства

1. Установи плагин, введи **те же** данные хранилища, разблокируй **той же**
   passphrase.
2. **Sync now** → Syncrypt скачает и расшифрует хранилище.

Готово. Дальше синхронизация происходит сама: чуть позже того, как ты
перестал печатать, при старте приложения, по возможности при закрытии/уходе в
фон — и всегда по **Sync now**.

## Чтение лога синхронизации

Каждое действие объяснено одной строкой, например:

```
Projects/ATM.md          удалённая версия новее         → скачано
Daily/2026-07-16.md      локальный файл изменился       → загружено
Old/Deprecated.md        помечен удалённым в манифесте  → удалён локально (в корзину)
Ideas.md                 изменён с обеих сторон         → КОНФЛИКТ (см. conflicted copy)
```

Открой лог командой **Syncrypt: Show sync log**. Если синхронизация собирается
удалить или перезаписать много файлов сразу — она остановится и покажет полный
список; прочитай его перед подтверждением.
