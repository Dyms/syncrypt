## Summary

One paragraph: what this proposes and why it matters.

## Motivation

What problem does this solve? What happens if we do nothing? Which requirements
(see RFC-0002) does it serve?

## Guide-level explanation

Explain the proposal as if teaching it to a user or a new contributor. Use
examples and diagrams. Prefer concrete scenarios.

## Reference-level explanation

The precise design: data structures, algorithms, formats, interfaces, error
states. Enough for an implementer (human or AI) to build it without further
design decisions.

## Drawbacks

## Alternatives considered
